Uses Firebase Realtime Database (free forever). Takes 3 minutes to set up once — then both of you just open the same HTML file.

One-time setup:
1. Go to console.firebase.google.com → Add project
2. Skip Google Analytics → Create project
3. Left menu → Build → Realtime Database → Create database
4. Choose any region → Start in test mode → Enable
5. Copy the database URL — looks like:
https://your-app-default-rtdb.firebaseio.com
6. Paste it below → Save & Sync → share this HTML file with your partner
Leave the secret blank if you chose test mode (recommended).
